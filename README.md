# Cobby

Caddy with the [caddy-l4](https://github.com/mholt/caddy-l4) layer-4 module.

Built from the official `caddy:2.11.3-alpine` image with `xcaddy`. Contains only
Caddy + caddy-l4. No other modifications.

## Usage

```bash
docker pull ghcr.io/dobbyvpn/cobby:2.11.3
```

## Image tags

- `:2.11.3` — pinned to Caddy 2.11.3
- `:latest` — latest build from main
